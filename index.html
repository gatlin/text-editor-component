Text Editor Component

This textarea component is able to render line numbers using pure CSS. Further, - accessibility tools will pick it up as a simple, plain <textarea> ; - complete control over text rendering (syntax highlighting should be straightforward); - the text content is readily available as the `value` attribute; - any text between <text-editor> and </text-editor> will be used as the initial field value; - behaves well on mobile. Using it in your page is as simple as <body> <text-editor> Edit me! </text-editor> <script src="./text-editor.component.js" type="module"></script> </body>
